# OLED PC Monitor

Display PC stats on an i2c OLED screen connected to an arduino.

## Features

- CPU usage
- RAM usage
- More features coming soon

## Setup

1. Upload Arduino sketch
2. Install .NET 10
3. Run PC C# script with Arduino usb port (Example: dotnet run com5)

## Wiring

| OLED | Arduino |  
| VCC  | 5V |  
| GND  | GND |  
| SDA  | A4 |  
| SCL  | A5 |
